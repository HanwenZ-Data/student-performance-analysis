# Student Performance Analysis

## Overview
This project analyzes student performance across three subjects: Math, English, and Economics.

## Tools Used
- Python
- Pandas
- Matplotlib

## Project Structure
- data/ → dataset
- notebooks/ → analysis notebook

## Key Insights
- Student G has the highest overall performance.
- Student F shows the lowest scores.
- Score variation is highest in Math.

## How to Run
1. Open notebook in VS Code
2. Run all cells
